Integrantes do Grupo

Breve Descrição
Projeto educativo que explora como HTML, Banco de Dados e Aplicativos Mobile trabalham juntos em sistemas reais. Através de exemplos práticos como Uber Eats e Spotify, o site demonstra a arquitetura completa de uma aplicação moderna, abordando as três camadas fundamentais: apresentação, lógica e dados.

Tecnologias Utilizadas
- HTML5 - Estrutura semântica
- CSS3 - Design responsivo com Flexbox e Grid
- JavaScript - Interatividade e tabs dinâmicas
- GitHub - Controle de versão
